# Marketplace

Vienkāršs Laravel 12 projekts ar Blade un Bootstrap 5.

## Lapas

- **Sākumlapa** — hero sadaļa, navigācija, informācijas kartītes, kājene
- **Pieteikšanās** — pieteikšanās forma ar paroles parādīšanas/paslēpšanas pogu
- **Reģistrācija** — reģistrācijas forma ar vārdu, e-pastu, paroli un apstiprinājumu

## Tehnoloģijas

- Laravel 12
- Blade
- Bootstrap 5.3

## Palaišana

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan serve
```
